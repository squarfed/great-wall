# Great Wall

## Prerequisites

You need to have the [xaos](https://xaos-project.github.io/) fractal renderer installed.

## Build instructions

The Rust toolchain must be installed:

```sh
cargo build
```

and run with

```
cargo run
```

## Nix configuration

If you are using the nix package manager you can create a build environment with:

```sh
nix develop
```
